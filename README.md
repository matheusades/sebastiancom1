⚙️ Como rodar o projeto
1. Clone o repositório
git clone https://github.com/matheusades/sebastiancom1.git
2. Acesse a pasta
cd sebastiancom1
3. Instale as dependências
npm install

ou

pip install -r requirements.txt
4. Execute o projeto
npm run dev

ou
